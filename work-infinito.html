REALISTI.CO

Realisti.co is a startup for real estate virtual tours, involving IoT communication, VR and 360 Technologies

As a software engineer, I have developed the entire mobile application for Apple devices. I decided to use the MVVM pattern for this app, and to use a reactive style approach using RxSwift and related libraries. The project involved various interesting and at the same time very challenging parts. Firstly, the connection and use of 360-degree cameras using Google's OSC protocol, for which I created a connection framework from scratch and always using the reactive programming paradigm. Secondly, the visualization of the 360-degree images for the preview of the photos, so I used third-party libraries modeling them to be similar to the structure of the application. Lastly, and probably the most complex problem to solve, the synchronization between the data on the device and the backend. This is due to various business logic and the need to avoid any simultaneous change by different users on the same buildings. Problem solved after some brain storming and several days of development and testing.
